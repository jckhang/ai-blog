# Ferret-UI苹果移动界面理解

## 亮点摘要


**Apple研究者** 开发 **Ferret-UI**，专为移动UI屏幕理解优化的多模态大模型：

- **指代（Referring）**: 理解用户对UI元素的自然语言指代
- **定位（Grounding）**: 将语言描述映射到具体界面组件
- **推理（Reasoning）**: 理解应用功能与交互流程

**意义**: 这是**设备端Agent**的关键组件 —— 让AI直接理解手机屏幕，实现"自动点击""自动填写"等操作，无需API对接。
