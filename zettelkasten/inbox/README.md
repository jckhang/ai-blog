# Inbox 文件夹说明

这是**闪念笔记 (Fleeting Notes)** 的存放位置。

## 用途

- 快速捕捉灵感、想法、听到的要点
- 无需完整结构，一句话即可
- 不必立即完善，只需记录

## 处理原则

- ⏱️ **24-48小时内**必须处理：转为文献笔记或永久笔记
- 🗑️ 过期未处理的内容**直接删除**
- ❌ 不要长期保留在 inbox！

## 命名

```
任意文件名.md
# 或按日期
2026-02-19-1234-some-idea.md
```

## 模板（可选）

```markdown
---
id: 临时ID
created: YYYY-MM-DD
source: （如有）
---

原始想法/灵感...

待处理：[ ] 转为文献笔记 [ ] 转为永久笔记 [ ] 丢弃
```

## 工作流

1. 有灵感 → 创建文件到 `inbox/`
2. 每天固定时间（如晚上）回顾 inbox
3. 决定每张笔记的命运：
   - 有价值 → 提取核心 → 制作文献笔记或永久笔记
   - 无价值 → 删除
4. 清空 inbox（目标是保持 0-5 条）

---

**记住**: Inbox 是**临时缓冲区**，不是存储位置。信任系统：记下来就忘记，但记得处理！
